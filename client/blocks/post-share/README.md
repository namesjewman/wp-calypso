# Post Share

Component/Block to share posts to social-media accounts

## Example

```es6
import PostShare from 'blocks/post-share';

<PostShare
	post={ post }
	siteId={ siteId } />
```

## Props

### `post`

A post object.

### `siteId`

The id of the site

### `disabled`

If it's true, the form elements of the component will be disabled such as buttons, input fields, etc.
